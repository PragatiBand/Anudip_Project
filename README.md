Trip Booking System

This project is a console-based Trip Booking System developed using Java and MySQL. It allows users to manage flight, hotel, and bus bookings through a simple menu-driven interface. The system uses JDBC for database connectivity and performs CRUD operations such as add, view, update, delete, and search bookings. Each booking module is well-structured and user-friendly. Overall, the project demonstrates effective use of Java, database integration, and real-world travel booking functionality.
